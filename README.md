### 新员工考试
小聪入职新公司，参加线上的新员工必备考试，考试共25题，以此是10个判断题（每题2分）、10个单选题（每题4分）和5个多选题（每题8分），总分100分。
考题只能顺序作答，答对题目获得相应分数，答错题目获得0分，考试系统不提示作答是否正确，答题过程中如果累积有3题答错，直接中止考试并计算考试分数。
小聪考试结果是N分（0<=N<=100），请根据小聪的分数，算出所有可能的答题情况的个数。
